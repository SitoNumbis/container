# Sito Container

Super flexible container, very similar to @mui's Box, but defaults to flex

### How to use it?

```
import SitoContainer from "sito-container";

function App() {
    return {
        <SitoContainer>
            Hola Mundo
        </SitoContainer>
    }
}

```
